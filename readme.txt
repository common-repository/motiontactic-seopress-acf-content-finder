== Description ==

Plugin will loop through ACF fields on a post or page adding all text areas to SEOPress content for analysis.

The plugin will add all of these fields to the SEOPress content for analysis:

*   Text
*   Text Area
*   WYSIWYG
*   Email

The plugin will dive into these field types to fine sub fields.

*   Flexible Content
*   Repeater
*   Group
*   Email

== Frequently Asked Questions ==

= What plugins do I need? =

This plugin requires you are using SEOPress and ACF

= Do I need ACF Pro? =

Nope, but we do work with fields that are included in ACF Pro.
